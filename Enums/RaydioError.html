RaydioError

public enum RaydioError : Equatable, Error

Represents an error state of the SDK Most of the error states can be gracefully processed

  • NOT_SUBSCRIBED

    We are not subscribed to the backend, meaning all requests that we send are just rejected. This could happen if the sdk is not connected or in the disconnected state A connect call should fix this

    Declaration

    Swift

    case NOT_SUBSCRIBED
  • STREAM_NOT_FOUND(streamId:)

    We tried to play/pause a stream that is not existing anymore This could happen for several reasons, all of them involve network issues on either the streamer or listener side A potential workflow could be to request the current streams from the backend again, and if the List contains the clicked streamId play it and if not display a message

    Declaration

    Swift

    case STREAM_NOT_FOUND(streamId: String)
  • LISTEN_FAILED(streamId:)

    Listening failed for the stream. In the Log or ListenStageDelegate more information should be provided why. The problem could be that the streamer closed during connection the stream or that network issues occurred. The same solution as for STREAM_NOT_FOUND could be applied

    Declaration

    Swift

    case LISTEN_FAILED(streamId: String)
  • INTERNAL(description:)

    Something internally went wrong and the logs could provide more information. There is no recovery possible of the action and just an information could be displayed

    Declaration

    Swift

    case INTERNAL(description: String)
