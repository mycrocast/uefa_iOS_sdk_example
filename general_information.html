General Information

This document contains a few general information which are not specific to the Android or iOS SDK but should help testing and highlight some current shortcomings.

How to start a stream

Login

Visit www.mycrocast.de and in the upper tab select “Streamer” and login with the credentials provided.

Start stream

Hit the stream start button and

  • Select a genre

  • Select a language

  • Select Event

  • Enter a title

  • Select a microphone

  • Hit conintue -> you are live a few moments later

The flow will be simplified in the near future (see current shortcomings)

Use audio file as content

You can instead of your own voice use an audio file that is than broadcasted.

To do so:

  • Drag and drop a file into the upper right corner of the webseit

  • Hit the play button on the entry that appeared there

How to create multiple entries in the same group

You need to create multiple streams, you need to log in with different users in different tabs.

Each stream must have the EXACT same title.

A future update will simplify this process.

Current shortcomings

MatchId

Currently the provided matchId is always just “0”. This will change in the future update for our webplatform and backend but does not require any updates from the SDK

Group creation

Currently to make a stream with multiple entries in the same stream, the title must match exactly. See start stream above.

An update from our frontend will change the stream start process where the streamer just needs to select the language and game. This will follow shortly and again does not require any updates to the SDK.
